## 📑프로젝트 주요 기능


- 게시판 : 게시물/댓글 CRUD 기능, 조회수, 페이징 및 검색 처리
- 회원 : Security 회원가입 및 로그인, 회원정보 수정, 회원가입 시 유효성 검사 및 중복 검사
- 관리자 : 회원 리스트 조회 
- 추가 기능 : 게시물 이미지 첨부 기능, 게시물 신고하기, 찜하기(위시리스트)
